# Procurement Performance Dashboard | Power BI

## Project Overview
This interactive Power BI dashboard analyzes procurement performance using key procurement KPIs, supplier performance metrics, cost savings, and compliance analysis.

## Objectives
- Monitor procurement spending
- Evaluate supplier performance
- Track compliance metrics
- Identify cost-saving opportunities
- Support procurement decision-making

## Tools Used
- Power BI
- Power Query
- DAX
- Microsoft Excel / CSV

## Dashboard Pages
### Procurement Overview
- Total Spend
- Total Savings
- Purchase Orders
- Supplier Count
- Procurement Trend

### Supplier Performance
- Supplier Performance Score
- Cost Savings by Supplier
- Defective Units
- Delivery Performance
